# mp-rs
## Description :

a simple TUI music player written in rust
(still in early development)
## How to run it ? :

- to run the project run the following commandes in your terminal :
```
git clone https://github.com/ihab65/mp-rs.git
cd mp-rs/
cargo run
```
## Navigation :
| key         | action      |
| ----------- | ----------- |
| q           | exit        |
| Enter       | run         |
| Space       | pause/play  |
